# Babullee 李仁柏 — Portfolio

Personal portfolio website for Babullee, a Grade 10 student roboticist, programmer, strategist, and member of VEX Team 10004X.

## Highlights

- 2025 VEX Taiwan Open
- VEX World Championship competitor
- Team 10004X — 11th in qualification matches
- C++ and VEXcode
- Robotics programming, engineering documentation, strategy, and leadership
- Mathematics card-game development

## Technology

React, TypeScript, Tailwind CSS, Vinext, and Vite.

## Development

```bash
npm install
npm run dev
```

Create a production build with:

```bash
npm run build
```

The public portfolio currently uses placeholders for photographs, match videos, notebook pages, code samples, the mathematics-game link, and the final résumé.
